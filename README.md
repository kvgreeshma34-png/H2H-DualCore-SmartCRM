# SmartCRM - AI-Powered Customer Management Portal

## 1. Project Title & Tagline
**SmartCRM** - Offline-first customer management with AI-driven insights for crisis situations

## 2. Problem Statement
During power cuts, network outages, or conflicts, businesses lose access to customer data. Critical customer needs go untracked. Emergency responders can't prioritize effectively.

## 3. Proposed Solution
A single HTML file that works completely offline, stores data locally, and uses AI insights to help businesses manage customers during network outages.

## 4. Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- LocalStorage API for offline persistence
- Chart.js for analytics dashboard
- No external dependencies - 100% offline capable

## 5. Features
- Add/Edit/Delete customers with priority levels (Normal/High/Critical)
- AI-driven insights: follow-up alerts, priority flags, health scoring
- Works completely offline (data persists in browser)
- Export customer data as CSV/Excel and JSON
- Dark mode toggle for night use
- Analytics dashboard with bar charts
- Search functionality
- Health score for each customer (0-100%)

## 6. Architecture / Flow
User opens index.html → Browser loads app → Data from localStorage → Display customers & AI insights → User actions → Update localStorage → Export to CSV/JSON for backup

No backend, no APIs - pure client-side solution for offline-first operation.

## 7. Setup Instructions
1. Download index.html from this repository
2. Open in ANY web browser (Chrome, Firefox, Edge, Safari)
3. No installation, no internet required
4. Start adding customers immediately
5. Data automatically saves offline
